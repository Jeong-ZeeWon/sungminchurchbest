# 성경 연구 프롬프트

이 폴더는 성경 본문 연구에 사용하는 프롬프트를 보관합니다.

## 기본 원칙

앞으로 사용자가 `성경 연구 프롬프트 전부`, `전부 돌려줘`, `통합으로 해줘`라고 말하면 기본값은 아래 통합 연구 프롬프트 묶음입니다.

```text
prompts/bible-study/integrated/00-integrated-bible-study-all.md
```

## 통합 연구 프롬프트

통합 연구 프롬프트는 노션의 `통합 연구 프롬프트` 페이지를 기준으로 합니다.

원본: https://www.notion.so/2e05e604d8c681df9ea2c680a75bae61

## 실행 흐름

1. 원어 연구
2. 구조 연구
3. 신학 비교
4. 본문 비평
5. 비판적 사회 윤리
6. 묵상 목회 적용
7. 다라쉬 연구
8. 신학자 대화식 해석
9. 주석 통합
10. 예화 탐색
11. 설교 준비 통합

## 저장 원칙

연구 결과는 기본적으로 `Jeong-ZeeWon/bible-research` 저장소에 본문별 폴더로 저장합니다.
