# 설교 준비 통합 프롬프트

원본 Notion: https://www.notion.so/2e05e604d8c6817da9b0f3258f1d0b16

## 목적
성경 본문 연구 결과를 실제 설교 준비로 연결한다. 본문 중심, 회중 맞춤, 복음 중심, 실천 가능한 설교 구성을 돕는다.

## 입력값
- 설교 본문:
- 설교 상황:
- 회중 특성:
- 설교 목표:
- 설교 시간:

## 준비 단계

### 1. 본문 선정 및 기초 분석

- 본문 범위
- 문맥
- 회중 상황
- 설교 목표

### 2. 종합적 본문 연구

- 원어 연구
- 문맥 연구
- 구조 분석
- 주석 비교
- 신학적 분석
- 사회문화 배경
- 성경 간 비교

### 3. 핵심 메시지 도출

- 중심 주제 Big Idea
- 신학적 원리
- 복음 연결점

### 4. 회중 분석 및 적용점 개발

- 영적 필요
- 정서적 필요
- 실제적 필요
- 개인, 관계, 사회, 영적 차원의 적용

### 5. 설교 구조 설계

- 제목
- 도입
- 본론 대지
- 예화와 설명
- 적용
- 결론
- 기도

### 6. 예화 개발

- 성경적 예화
- 역사적 예화
- 현대적 예화
- 개인/목회 현장 사례

### 7. 실천적 적용 구체화

SMART 적용법을 따른다.

- Specific: 구체적
- Measurable: 확인 가능
- Achievable: 실천 가능
- Relevant: 본문과 관련
- Time-bound: 기한 설정

### 8. 최종 점검

- 본문의 의도를 반영했는가?
- 복음 중심 메시지가 분명한가?
- 회중의 상황과 필요에 맞는가?
- 적용이 구체적인가?
- 설교 구조가 명확한가?
- 제한 시간 안에 전달 가능한가?

## 출력 형식

```markdown
# 설교 준비 통합

## 1. 본문과 상황
## 2. 본문 연구 요약
## 3. 중심 주제
## 4. 복음 연결점
## 5. 회중 분석
## 6. 설교 개요
## 7. 예화와 적용
## 8. 기도 제목
## 9. 최종 점검
```

## 유의사항
- 연구자료를 설교문으로 성급히 바꾸지 말고, 설교 준비 단계로 정리한다.
- 본문 중심성과 회중 적합성을 함께 붙든다.
- 복음 연결은 억지로 붙이지 않고 본문 흐름 안에서 도출한다.
